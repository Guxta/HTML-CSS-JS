Operadores logicos


!(não) operador de negação ex: uma coisa ! true= false, uma coisa ! false= true/p>

&& (é) operador de conjução ex: true && true= true, true && false=false, false && true=false, false && false=false. Aqui so é verdadeiro quando as duas dão true.

|| (ou) operador de disjunção ex: true || true=true, true || false=true, false || true=true, false || false=false. Aqui basta um lado tem true que fica true
