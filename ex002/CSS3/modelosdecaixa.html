Modelos de caixa

Toda palavra escrita fica dentro de um retangulo, que pode ser modificada usando: height (altura), width(largura), border(borda da caixa),padding(preenchimento), margin(margen), outline(contorno)

Tipos de caixas

box-level é quando dentro de uma caixa se cria outra caixa porem para baixo ocupando toda a largura ex: tag div

inline-leve é quando dentro de uma caixa se cria outra porem ela vai para o lado e n ocupa toda o espaço ex: tag span
